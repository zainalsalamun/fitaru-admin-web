# Fitaru Admin CMS

Next.js admin dashboard for managing the Fitaru app.

## Planned Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- TanStack Query
- TanStack Table
- React Hook Form
- Zod
- Recharts
- Lucide React
- Supabase

## Planned Pages

```text
/login
/
/users
/content
/food-database
/exercise-database
/feedback
/settings
/notifications
```
